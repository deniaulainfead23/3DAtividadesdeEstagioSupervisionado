# Atividade 9 – Criando um Testador de Código em C com Apoio de IA

## Objetivo

Utilizar uma IA Generativa para criar, testar e corrigir algoritmos simples em linguagem C.

---

## Ferramentas

Use uma das opções:

- ChatGPT
- Claude
- Gemini

E para testar o código:

- OnlineGDB
- Replit
- Code::Blocks
- Dev-C++

---

## Desafio

Você deverá pedir para a IA criar um programa em C que teste automaticamente funções simples.

O programa deverá verificar:

- Dobro de um número
- Número par ou ímpar
- Média de dois valores
- Maior entre dois números

---

## Parte 1 – Prompt Inicial

Copie e cole na IA:

```text
Crie um programa em linguagem C para funcionar como testador de algoritmos simples.

O programa deve ter funções para:

1. Calcular o dobro de um número.
2. Verificar se um número é par.
3. Calcular a média de dois números.
4. Identificar o maior entre dois números.

Depois, crie testes automáticos no main para verificar se cada função está correta.

O programa deve mostrar mensagens como:
"Teste OK" ou "Teste com erro".

Use linguagem simples e comente o código para estudantes iniciantes.
```

---

## Parte 2 – Testando o Código

1. Copie o código gerado pela IA.
2. Cole no OnlineGDB ou outro compilador C.
3. Execute o programa.
4. Verifique se apareceu mensagem de erro ou se todos os testes passaram.

---

## Parte 3 – Corrigindo com IA

Se o código apresentar erro, copie a mensagem do compilador e envie para a IA com o prompt:

```text
Meu código em C apresentou este erro:

[cole aqui a mensagem de erro]

Explique o erro de forma simples e corrija o código.
```

---

## Parte 4 – Melhorando o Testador

Peça para a IA melhorar o programa:

```text
Melhore o testador de algoritmos em C adicionando:

- Contador de testes corretos
- Contador de testes com erro
- Resultado final em porcentagem
- Mensagem final de desempenho do aluno
```

---

## Parte 5 – Desafio Criativo

Agora solicite:

```text
Transforme esse programa em um mini sistema de avaliação de algoritmos em C para alunos iniciantes, com menu de opções e testes automáticos.
```
