🎮 IA Prática 2 – Mini Campo Minado
Objetivo

Desenvolver uma versão simplificada do clássico Campo Minado utilizando HTML, CSS e JavaScript.

Ao concluir esta atividade, o aluno será capaz de:

Construir interfaces utilizando HTML;
Aplicar estilos com CSS;
Manipular elementos da página com JavaScript;
Utilizar matrizes para representar um tabuleiro;
Trabalhar com estruturas de repetição;
Utilizar eventos de clique;
Alterar classes CSS dinamicamente.
O que será desenvolvido

O aluno criará um jogo contendo:

🎮 Tabuleiro 5×5
💣 Cinco minas distribuídas aleatoriamente
😀 Contador de casas abertas
😵 Tela de Game Over
🏆 Tela de Vitória
🔄 Botão Reiniciar

IA apenas para melhorar

O jogo deverá funcionar completamente antes do uso da IA.

Depois disso, o aluno poderá solicitar pequenas melhorias, como:

⏱️ adicionar cronômetro;
😊 criar níveis Fácil, Médio e Difícil;
🚩 adicionar bandeiras para marcar minas;
🔊 inserir sons simples;
🎨 melhorar as cores do tabuleiro;
✨ criar pequenas animações;
💾 salvar o recorde usando localStorage.
Exemplo de prompt
Melhore este jogo de Campo Minado desenvolvido em HTML, CSS e JavaScript.

Não altere sua estrutura principal.

Implemente apenas:

- cronômetro;
- bandeiras;
- mudança de cores;
- sons simples;
- comentários explicando o código.

Mantenha o projeto adequado para estudantes iniciantes.
