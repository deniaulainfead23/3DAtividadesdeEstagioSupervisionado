Letramento em Inteligência Artificial para Programação

Tema: Aprendendo a conversar com a IA para desenvolver software de forma responsável.

Objetivos

Ao final da atividade o aluno deverá ser capaz de:

compreender o que é IA Generativa;
diferenciar IA de mecanismo de busca;
escrever prompts mais eficientes;
validar o código produzido pela IA;
identificar erros e limitações;
reconhecer "alucinações" da IA;
documentar o uso da IA em um projeto.
O que será ensinado?
📚 Módulo 1 – A IA não substitui o programador

Explicar que a IA:

✅ gera código

✅ explica código

✅ corrige erros

✅ documenta projetos

Mas ela não sabe se aquele código funciona no projeto do aluno.

📚 Módulo 2 – Anatomia de um bom prompt

Mostrar a diferença entre:

❌

Faça um sistema.

e

✅

Crie um jogo em HTML, CSS e JavaScript para iniciantes contendo apenas três arquivos: index.html, style.css e script.js.

O prompt deve conter:

contexto;
objetivo;
tecnologias;
restrições;
nível do aluno;
resultado esperado.
