# Atividade 7 – Crie Seu Próprio Jogo com IA

## 🎮 Objetivo

Utilizar o Claude para criar um jogo simples e interativo sem necessidade de conhecimentos avançados de programação.

---

## 🚀 Desafio

Você foi contratado por uma empresa de jogos para criar um mini game educativo para estudantes.

Seu objetivo é utilizar IA para desenvolver um jogo funcional que possa ser executado diretamente no navegador.

---

## Parte 1 – Acessando o Claude

Acesse:

```text
https://claude.ai
```

Faça login utilizando uma conta gratuita.

---

## Parte 2 – Criando o Jogo

Copie e cole o prompt abaixo:

```text
Você é um desenvolvedor de jogos.

Crie um jogo completo em HTML, CSS e JavaScript que funcione em um único arquivo.

Tema: Quiz de Tecnologia.

Requisitos:

- 5 perguntas sobre tecnologia
- Sistema de pontuação
- Botão de reiniciar
- Design moderno
- Funcionar sem internet
- Código completo pronto para salvar como HTML
```

---

## Parte 3 – Testando o Jogo

1. Copie o código gerado.
2. Abra o Bloco de Notas.
3. Cole o código.
4. Salve como:

```text
quiz.html
```

5. Abra o arquivo no navegador.

---

## Parte 4 – Personalizando

Peça ao Claude:

```text
Transforme o jogo em um quiz sobre meu curso.
```

ou

```text
Transforme o jogo em um quiz sobre Inteligência Artificial.
```

ou

```text
Transforme o jogo em um quiz sobre redes de computadores.
```

---

## Parte 5 – Evoluindo o Projeto

Agora solicite:

```text
Adicione:

- Sons
- Cronômetro
- Ranking de pontuação
- Efeitos visuais
- Tela inicial
```

---

## 🎯 Desafio Extra

Solicite ao Claude:

```text
Transforme este quiz em um jogo estilo Show do Milhão.
```

ou

```text
Transforme este quiz em um jogo de aventura com escolhas.
```
