> 🚀 **Desafio:** Será que uma IA consegue criar um aplicativo funcional a partir de uma descrição? Vamos descobrir!

---

## 🎯 Objetivo

Utilizar uma IA Generativa para criar o protótipo de um aplicativo sem necessidade de conhecimento avançado em programação.

---

## 🛠 Ferramentas

Escolha uma das opções:

- ChatGPT
- Claude
- Gemini

---

## 📱 Desafio

Você deverá solicitar que a IA crie um aplicativo simples para resolver um problema do dia a dia.

Exemplos:

- Organizador de estudos
- Lista de tarefas
- Controle de gastos
- Agenda escolar
- Calculadora de médias
- Controle de livros lidos

---

## 🚀 Parte 1 – Criando o Aplicativo

Copie e cole o prompt abaixo:

```text
Você é um desenvolvedor de software.

Crie um aplicativo em HTML, CSS e JavaScript para ajudar estudantes.

O aplicativo deve possuir:

- Cadastro de tarefas
- Lista de atividades
- Marcação de tarefas concluídas
- Interface moderna
- Funcionar em um único arquivo HTML

Forneça o código completo.
```

---

## 💻 Parte 2 – Testando

1. Copie o código gerado.
2. Abra o Bloco de Notas.
3. Cole o código.
4. Salve como:

```text
app.html
```

5. Abra o arquivo no navegador.

---

## 🎨 Parte 3 – Personalização

Peça à IA:

```text
Melhore o aplicativo adicionando:

- Tema escuro
- Barra de progresso
- Contador de tarefas concluídas
- Ícones
- Design moderno
```

---

## 🧠 Parte 4 – Avaliação da IA

Responda:

- O aplicativo funcionou?
- A IA criou tudo corretamente?
- Foi necessário corrigir algo?
- O resultado ficou útil?

---

## ⭐ Desafio Extra

Peça para a IA:

```text
Transforme este aplicativo em uma versão para celular com visual profissional.
```

ou

```text
Adicione um sistema de login simples.
```
