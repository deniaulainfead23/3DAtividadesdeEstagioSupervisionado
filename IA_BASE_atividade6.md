# Atividade 5 – Automação com IA utilizando Claude

## Objetivo

Utilizar o Claude para criar uma automação simples que auxilie na organização dos estudos e das atividades escolares.

---

## O que você irá construir?

Um organizador inteligente de estudos contendo:

- Lista de tarefas
- Prioridade
- Prazo
- Tempo estimado
- Ordem recomendada de execução

---

## Parte 1 – Acessando o Claude

1. Acesse o site:

```text
https://claude.ai
```

2. Faça login utilizando uma conta gratuita.

---

## Parte 2 – Criando a Automação

Copie e cole o prompt abaixo no Claude:

```text
Você é um assistente de organização escolar.

Crie uma automação simples para organizar meus estudos da semana.

Tenho as seguintes tarefas:

1. Estudar lógica de programação
2. Fazer exercício de banco de dados
3. Revisar redes de computadores
4. Criar uma apresentação sobre Inteligência Artificial
5. Entregar atividade de estágio

Organize essas tarefas em uma tabela contendo:

- Tarefa
- Prioridade
- Prazo sugerido
- Tempo estimado
- Ordem recomendada

Após a tabela, gere uma página HTML simples para visualizar essas informações no navegador.
```

---

## Parte 3 – Análise da Resposta

Após receber a resposta do Claude, responda:

- A tabela foi organizada corretamente?
- As prioridades fazem sentido?
- Os prazos são realistas?
- A ordem das tarefas parece adequada?

---

## Parte 4 – Testando o HTML

1. Copie o código HTML gerado pelo Claude.
2. Abra o Bloco de Notas.
3. Cole o código.
4. Salve o arquivo com o nome:

```text
organizador.html
```

5. Abra o arquivo no navegador.

---

## Parte 5 – Melhorando a Automação

Solicite ao Claude:

```text
Melhore esse organizador adicionando:

- Cores
- Campo para marcar tarefas concluídas
- Barra de progresso
- Design moderno para estudantes
```

Compare o resultado com a versão anterior.

---

## Desafio Extra

Peça ao Claude para transformar o organizador em uma aplicação mais completa:

```text
Transforme este organizador em um sistema de gerenciamento de estudos com:

- Cadastro de tarefas
- Prioridade automática
- Contador de tarefas concluídas
- Barra de progresso
- Layout responsivo
```
