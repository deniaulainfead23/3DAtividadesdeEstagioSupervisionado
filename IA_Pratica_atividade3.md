# 📚 O que é o Jogo da Memória?

O objetivo é encontrar todos os pares de cartas iguais.

Cada clique revela uma carta.

Quando duas cartas iguais forem encontradas, elas permanecem abertas.

Caso sejam diferentes, elas voltam a ficar escondidas.

O jogo termina quando todos os pares forem encontrados.
